# Custom CRA Template

## Installation and Startup

    npx create-react-app [project-name] --template john
    npm start

## Dependencies Included

    • axios
    • depcheck
    • jest
    • node-sass
    • react
    • react-dom
    • react-router-dom
    • react-scripts
    • styled-components
